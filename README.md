## Introduction
This repository contains the codebase for scraping and analyzing news articles from _het Parool, NRC, Trouw_ and the _Volkskrant_. Data collection starts one month before the release of all parliamentary party platforms (22-08-2023), and continues until the month before the elections (22-10-2023).

## Installation

### Prerequisite requirements.
Python version 3.11.5.

### Cloning the GitHub repository and installing python dependencies.
Clone the repository:
```bash
git clone https://github.com/stuurpropje/sentiment-analysis
```
3. Navigate to the project root directory.
4. Install the required dependencies:
```bash
pip install -r requirements.txt
```
